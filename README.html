Nest framework TypeScript starter repository.

Googhe sheets link

Project setup

$ pnpm install

Generate migrations

$ pnpx prisma migrate dev --name init

Install and generate Prisma Client

$ pnpm install @prisma/client

Compile and run the project

# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod

Run tests

# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
